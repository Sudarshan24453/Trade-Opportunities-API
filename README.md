# Trade Opportunities API

## Overview
Trade Opportunities API is a FastAPI service that analyzes Indian market sectors and generates trade opportunity insights using AI.

The API collects recent sector news using web search, analyzes it using Google Gemini AI, and produces a structured markdown report with market insights and potential trade opportunities.

---

## Features

- FastAPI backend
- Google Gemini AI integration
- DuckDuckGo web search for sector news
- JWT authentication
- Rate limiting
- Input validation
- In-memory session tracking
- Markdown report generation

---

## Tech Stack

Backend: FastAPI  
AI Model: Google Gemini  
Web Search: DuckDuckGo Search  
Authentication: JWT  
Storage: In-memory storage
